# Tarkvaraarenduse-projekt-Salme-saaga-

## Eesmärk
Projekti eesmärk on luua viikingi mäng, mis arendab huvi ja teadmisi Eestis olnud viikingite ja nende ajaloo vastu. Teha õppetöö muuseumites ja koolides interaktiivsemaks. arendada oma oskusi.

## Mille raames projekt on valminud
DTI(Digitehnoloogia Instituut) + ELU projekt: Tulevikuõpe ja loovtehnoloogiad: millist lugu jutustavad mäluasutused?
